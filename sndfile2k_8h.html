◆ SF_FORMAT

enum SF_FORMAT

Set of format bit flags.

The SF_INFO::format field of the SF_INFO structure is made up of the bit-wise OR of a major format type (values between 0x10000 and 0x08000000), a minor format type (with values less than 0x10000) and an optional endianness value.

SF_FORMAT_SUBMASK, SF_FORMAT_TYPEMASK and SF_FORMAT_ENDMASK bit masks can be used to separate major, minor format types and endianness value.

Note
Not all combinations of endianness and major and minor file types are valid. You can use sf_format_check() function to check format flags combination.
